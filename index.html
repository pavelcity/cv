Такой подход позволил при создании моделей и админки предусмотреть загрузку контента для разных версий сайта. Также для каждой версии сайта используется свой набор css стилей и разный формат отображения контента. Главная цель этой задумки чтобы для мобильной версии не грузить стили для десктопа и ненужные html блоки из десктопной версии в мобильную.
